## ビューのフィルター設定方法

### 手順
1. テーブルを開く
2. 左上のビュー名をクリック
3. 設定したいビューを選択
4. フィルターアイコン（漏斗マーク）をクリック
5. 条件を設定
6. 保存

### よく使うフィルター条件

**今日のデータ**
```
日付 = Today
```

**今週のデータ**
```
日付 >= ThisWeek.Start
日付 <= ThisWeek.End
```

**今月のデータ**
```
日付 >= ThisMonth.Start
日付 <= ThisMonth.End
```

**空白のフィールド**
```
フィールド名 is empty
```

**複数条件（AND）**
```
ステータス = 進行中
AND
クリティカルパス = true
```

## ダッシュボードのカスタマイズ

### 列の表示/非表示
1. 列ヘッダーを右クリック
2. 「列を非表示」を選択
3. 必要な列だけ表示

### 列の並び替え
1. 列ヘッダーをドラッグ
2. 好きな位置にドロップ

### 行の高さ調整
1. 表示設定アイコンをクリック
2. 行の高さを選択（コンパクト/標準/広い）

### 色の設定
1. 列ヘッダーを右クリック
2. 「条件付き書式」を選択
3. 条件と色を設定

**おすすめ色設定**

| 条件 | 色 |
|------|-----|
| ステータス = 完了 | 緑 |
| ステータス = 進行中 | 青 |
| ステータス = 遅延 | 赤 |
| 判定 = 合格 | 緑 |
| 判定 = 要手直し | 赤 |

## ガントチャートの使い方

### 表示方法
1. `20_大工程`、`21_中工程`、または `22_小工程` を開く
2. 右上のビュー切替で「ガントチャート」を選択

### 操作方法
- **期間変更**: バーの端をドラッグ
- **移動**: バーをドラッグ
- **詳細表示**: バーをクリック

### 表示設定
- 日表示/週表示/月表示の切替可能
- 色はテーブルの「表示色」フィールドで設定

## レポート出力

### Excelエクスポート
1. ビューを表示
2. 右上の「...」メニューをクリック
3. 「エクスポート」→「Excel」を選択

### 印刷
1. ビューを表示
2. ブラウザの印刷機能（Ctrl+P / Cmd+P）

---

## トラブルシューティング

### ビューが表示されない
→ フィルター条件を確認（条件に合うデータがない可能性）

### 条件がうまく動かない
→ フィールドの型を確認（日付型、数値型など）

### 色が反映されない
→ ブラウザをリロード

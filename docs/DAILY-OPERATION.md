# 建設業版Lark Base - 日常業務マニュアル

> 現場担当者向け シンプル操作ガイド

---

## 1日のタイムライン

```
08:00  朝礼・KY活動 → 51_KY活動記録 に入力
08:30  作業開始
12:00  昼休憩
13:00  午後の作業開始
16:30  作業終了
17:00  日報入力 → 40_作業日報 に入力
17:15  退勤
```

---

## 朝やること（5分）

### KY活動記録の入力

1. **51_KY活動記録** を開く
2. 右上の `+` ボタンをタップ
3. 入力項目：

| 項目 | 入力例 |
|------|--------|
| 実施日 | 今日の日付（自動） |
| 作業班 | 躯体班 |
| 職長 | 山田太郎 |
| 参加人数 | 8 |
| 作業内容 | 3階柱配筋 |
| 想定される危険 | 1. 開口部からの墜落 2. 鉄筋による切傷 |
| 対策 | 1. 手すり設置確認 2. 革手袋着用 |
| 本日の目標 | 墜落ゼロ！ |

4. 保存

**ポイント**: 写真は不要。3分で終わらせる。

---

## 夕方やること（5分）

### 作業日報の入力

1. **40_作業日報** を開く
2. 右上の `+` ボタンをタップ
3. 入力項目：

| 項目 | 入力例 |
|------|--------|
| 日付 | 今日の日付（自動） |
| 天候 | 晴れ |
| 作業可否 | 通常作業 |
| 元請人員 | 5 |
| 協力会社人員 | 23 |
| 本日の作業内容 | 3階柱配筋完了、3階梁配筋着手 |
| 本日の進捗 | 柱配筋100%、梁配筋30% |
| 明日の予定 | 3階梁配筋継続 |
| 写真 | 添付する（必須！） |

4. 保存

**ポイント**: 写真は最低3枚（全景・作業中・完了）

---

## 週1回やること

### 安全パトロール（所長・主任）

1. **50_安全パトロール** を開く
2. 新規レコード作成
3. 評価項目を記入：

| 評価項目 | 基準 |
|---------|------|
| 整理整頓 | 資材・廃材が整理されているか |
| 安全通路 | 通路が確保されているか |
| 足場・開口部 | 手すり・ネット・養生があるか |
| 保護具着用 | ヘルメット・安全帯を着用しているか |
| 重機・車両 | 誘導員配置、接触防止措置があるか |

4. 指摘事項があれば記入
5. 是正期限を設定
6. 保存

---

## 検査時にやること

### 検査記録の入力

1. **60_検査記録** を開く
2. 新規レコード作成
3. 入力項目：

| 項目 | 入力例 |
|------|--------|
| 検査日 | 今日の日付 |
| 検査種別 | 自主検査 |
| 検査項目 | 3階柱配筋検査 |
| 検査箇所 | 3階 C1〜C12柱 |
| 検査基準 | 配筋図通り、かぶり厚40mm以上 |
| 測定値 | かぶり厚45mm（最小） |
| 判定 | 合格 |
| 写真 | 検査状況写真（必須！） |

4. 保存

### 不合格の場合

1. 判定を「要手直し」に設定
2. 指摘事項を記入
3. 是正完了後、同じレコードに：
   - 是正内容を記入
   - 是正完了日を入力
   - 是正後の写真を添付

---

## 写真の撮り方

### 基本ルール
- **黒板を入れる**（工事名・日付・部位）
- **近景・中景・遠景**の3パターン
- **逆光を避ける**

### 日報用（3枚以上）
1. 全景（現場全体がわかる）
2. 作業状況（人が写っている）
3. 完了状況（出来形がわかる）

### 検査用（5枚以上）
1. 全景（検査対象の位置がわかる）
2. 近景（詳細がわかる）
3. 測定状況（測定器具を当てている）
4. 測定値（スケールの目盛りが読める）
5. 黒板アップ（情報が読める）

---

## よくあるミスと対策

### 入力し忘れた
→ 翌日でも入力する（日付は正しく）
→ 備考に「翌日入力」と記載

### 写真を撮り忘れた
→ 撮り直しが可能なら撮り直す
→ 不可能なら備考に理由を記載

### 選択肢にない項目
→ 「その他」を選択
→ 備考に詳細を記載
→ 管理者に選択肢追加を依頼

### 間違えて入力した
→ レコードを編集して修正
→ Lark Baseは履歴が残るので安心

---

## スマホからの入力

1. Larkアプリを開く
2. 下部メニューから「Base」を選択
3. 「建設業工事管理システム」を選択
4. テーブルを選んで入力

**ポイント**:
- 現場で写真を撮ったらその場で日報に添付
- 後回しにしない！

---

## 困ったときの連絡先

- 操作方法がわからない → 所長に聞く
- システムエラー → 管理者に連絡
- 選択肢を追加したい → 管理者に依頼

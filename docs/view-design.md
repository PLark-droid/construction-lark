# 建設業向け Lark Base ビュー設計

## 目的
ITリテラシーが低い建設現場の人でも直感的に使えるビュー設計を提供する。

## Lark Base ビュータイプ

### 1. Grid View (グリッドビュー)
- 表形式のビュー
- すべての詳細データを確認・編集できる
- デフォルトビューとして最適

### 2. Kanban View (カンバンビュー)
- ステータス別にカード表示
- 進捗管理に最適
- ドラッグ&ドロップで状態変更可能

### 3. Gantt View (ガントビュー)
- タイムライン表示
- 工程管理・スケジュール管理に最適
- 日程の可視化

### 4. Gallery View (ギャラリービュー)
- 画像やファイルを大きく表示
- 写真管理に最適
- 安全パトロール記録などに活用

### 5. Calendar View (カレンダービュー)
- カレンダー形式で表示
- 日程管理に最適
- 作業日報、KY活動記録などに活用

## テーブル別最適ビュー設計

### 1. 工事契約 (tblzeXSOwQjTY5wt)

#### デフォルトビュー: Grid View「全工事一覧」
- すべての工事を一覧表示
- 契約番号、工事名、発注者、契約金額、ステータスを表示

#### ビュー2: Kanban View「工事進捗管理」
- ステータス列でグループ化（計画中 → 契約済 → 施工中 → 検査中 → 完了）
- 現場責任者がドラッグ&ドロップで状態更新
- カードには工事名、着工日、竣工予定日を表示

#### ビュー3: Grid View「施工中案件」
- ステータスが「施工中」の工事のみ表示
- フィルタ: ステータス = 施工中
- ソート: 竣工予定日の早い順

#### ビュー4: Grid View「完了予定（今月）」
- 今月竣工予定の工事を表示
- フィルタ: 竣工予定日が今月
- ソート: 竣工予定日の早い順

### 2. 大工程 (tbln82ijUjFqUHEe)

#### デフォルトビュー: Gantt View「工程スケジュール」
- 全工程をガントチャートで表示
- 開始日・終了日でタイムライン表示
- 工事別にグループ化

#### ビュー2: Grid View「全工程一覧」
- すべての工程を一覧表示
- 工事名、工程名、開始日、終了日、ステータス

#### ビュー3: Grid View「進行中工程」
- ステータスが「進行中」の工程のみ
- フィルタ: ステータス = 進行中

### 3. 中工程 (tbl9s3ZtsNZzncSl)

#### ビュー3: Grid View「全中工程一覧」

### 4. 小工程 (tblM4zC4WQJTzx8Q)

### 5. 人員配置 (tblLQbNfEB6Bbimr)

## 現場で使いやすくするポイント

### 1. シンプルな命名
- ビュー名は短く、わかりやすく
- 「全○○一覧」「今週の○○」「△△別」など統一パターン

### 2. デフォルトビューの工夫
- 最も頻繁に使うビューをデフォルトに
- マスタ系 → Grid View
- スケジュール系 → Gantt View
- 進捗管理系 → Kanban View

### 3. フィルタとソートの活用
- 「今週」「今月」「施工中」など、よく使う条件をビューに設定
- ソートで優先度の高い情報を上位表示

### 4. グループ化の活用
- 関連データをまとめて表示
- 工事別、担当者別、ステータス別など

### 5. 視覚的なビューの活用
- 写真管理 → Gallery View
- 日程管理 → Calendar View
- スケジュール → Gantt View
- 進捗管理 → Kanban View

## ビュー作成の優先順位

### 優先度: 高
1. 工事契約 - Kanban View「工事進捗管理」
2. 大工程 - Gantt View「工程スケジュール」
3. 人員配置 - Gantt View「人員稼働スケジュール」
4. 安全パトロール - Gallery View「写真で確認」
5. 作業日報 - Calendar View「作業カレンダー」

### 優先度: 中
6. 機材配置 - Gantt View「機材使用スケジュール」
7. 中工程 - Gantt View「中工程スケジュール」
8. 小工程 - Kanban View「小工程タスクボード」
9. KY活動記録 - Calendar View「KYカレンダー」
10. 安全パトロール - Kanban View「是正状況管理」

### 優先度: 低
11. その他のGrid View（フィルタ・ソート設定）
